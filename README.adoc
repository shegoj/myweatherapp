
= Simple weather app

== About the app

The app provides weather information about a geographical location in the world.


== App Requirements
```
Java 1.8
Maven
Git
```
== To run the app

```
git clone https://github.com/shegoj/myweatherapp.git
cd myweatherapp
mvn spring-boot:run -Dapp.weather.locations=Nigeria/Lagos,Nigeria/Kano
